JLab Talk GO Review & Specifications

$49.00

Overview: Leveraging JLab's reputation in accessible consumer audio, the Talk GO offers multiple directional patterns. It allows users to switch between Cardioid and Omnidirectional modes on the fly.

Key Features & Technical Specs

  • Two directional patterns (Cardioid, Omnidirectional)
  • Quick-mute button with light ring indicator
  • 96kHz/24BIT recording capability
  • 5/8-inch standard sizing for mounting

Pros

  • Omni mode is great for capturing multiple voices in a room
  • Clean, highly visible mute indicator
  • Very well constructed and sturdy base

Cons

  • Pushes right up against the $50 threshold
  • Omni mode inevitably captures a lot of room echo

Should you actually buy it?

If you are multi-person podcasters, interviewers, and conference room users needing multidirectional capability., I think the JLab Talk GO is a solid grab for your setup. I would definitely use this daily.
